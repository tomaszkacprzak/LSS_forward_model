# Installation:
first setup the environment (environment.yaml). The only missing dependency there is Baryonforge - please install it via 
pip install git+https://github.com/DhayaaAnbajagane/BaryonForge.git

Then you can simply do python setup.py install


Folks on NERSC: you can use my environment by doing:
source activate /global/cfs/cdirs/m5099/env/pyccl_env
